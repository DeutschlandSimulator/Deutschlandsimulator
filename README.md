# DeutschlandSimulator

Ein interaktives, datenbasiertes Werkzeug zur Veranschaulichung möglicher Auswirkungen politischer Entscheidungen auf Staatshaushalt, Wirtschaft, Sozialsysteme und Gesellschaft.

**Demo:** [deutschlandsimulator.replit.app](https://deutschlandsimulator.replit.app)  
**GitHub:** [github.com/DeutschlandSimulator](https://github.com/DeutschlandSimulator)

---

## Über das Projekt

Der Deutschlandsimulator ermöglicht es, politische Szenarien zu simulieren und ihre modellbasierten Auswirkungen auf zentrale Kennzahlen zu erkunden. Das Projekt richtet sich an interessierte Bürgerinnen und Bürger, Journalisten, Studierende und alle, die politische Zusammenhänge besser verstehen möchten.

Das Projekt ist parteipolitisch neutral und steht in keiner Verbindung zu Parteien, Behörden oder Interessenverbänden.

---

## Tech Stack

- **Frontend:** React + Vite + TypeScript
- **Styling:** Tailwind CSS v4
- **Charts:** Recharts
- **Routing:** Wouter
- **Animation:** Framer Motion
- **UI-Komponenten:** shadcn/ui + lucide-react
